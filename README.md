# How to use
- Visit the [website](https://ai-or-not-acm-qu.netlify.app/) or clone it and run it using `bun run dev`

## How to Play
- Player clicks play
- Player attempts to select the Real image out of the 2 shown each round out of the 10 rounds
- Player gets a point for each correct guess
- At the end of the game the Player can add his name to the leaderboard
